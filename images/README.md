# Images

Put your dashboard screenshot here as **`dashboard.png`** — the main README displays it.

To capture it: in Power BI Desktop, **File → Export → Export to PDF** (then crop), or simply take a full screenshot of the report page (Win + Shift + S) and save it as `dashboard.png` in this folder.

A short GIF (`dashboard.gif`) showing interactivity also works well — reference it in the main README the same way.
